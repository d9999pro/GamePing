# GamePing

## 脚本介绍
用于检测VPS到各游戏服务器之间的延迟.

Game list:

+ League of Legends

+ PlayerUnknown's Battlegrounds

+ Tom Clancy's Rainbow Six Siege

+ Apex Legends

## How to run

**Curl

````bash
bash <(curl -L -s https://raw.githubusercontent.com/d9999pro/GamePing/main/GamePing.sh)
````


````docker
docker run --rm --net=host d9999pro/GamePing
````
